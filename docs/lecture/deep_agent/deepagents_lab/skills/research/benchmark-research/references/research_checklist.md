# Research checklist

Capture the strongest available evidence under these headings:

## Comparable systems

- Existing agent systems or papers close to the proposed task
- Why each one is relevant as inspiration or comparison

## Recent scientific writing

- Recent papers, benchmark papers, or system papers that sharpen the proposal
- What concrete idea, setup, or evaluation choice each source contributes

## Candidate datasets, APIs, or environments

- Exact name
- Access path or homepage
- Why it fits the topic

## Benchmarks and metrics

- Exact benchmark or evaluation setup
- Main metrics
- What the metric says about success

## Feasibility notes

- Cost, access, rate limits, or setup friction
- Open questions that still need validation

## Source URLs

- Keep a clean list of the URLs actually used

## Source information for later references

- Title
- Source or organization
- Year if available
- URL
- Short note on why this source was actually used in the proposal
