# Scripts

This folder holds important scripts for automating work in our project.

## Usage

- `cd` into `scripts/` folder
- use [uv](https://docs.astral.sh/uv/getting-started/installation/) 
- run:

```sh
uv run <script-name> [OPTIONS]
```

## TravelPlanner Related Scripts

### Evaluation Scripts

- `tp-eval`

### Postprocessing Scripts

- `tp-parse`
- `tp-extract`
- `tp-combine`
- `tp-format-check`
